arr-it.de JSON-Tools

JSON extract for Excel Use

Enter a valid JSON document. Either an array or an Object with arrays as properties will be converted.
A single array will be converted into an unnamed table. An object with properties will be converted into a list of named tables - where the property-name will be used as the name for the respective table. Still only arrays can be converted to tables.
Everything else will not be converted, instead it will be printed as stringified JSON. Inside table-cells, sub-documents are stringified as well.
